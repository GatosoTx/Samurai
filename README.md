**PLANTIAMIENTO DE LA IDEA**
la idea era una pagina que reflejara la rebeldia que representa la banda la idea se plasmo en un wireframe de baja y alta fidelidad en los cuales se creo un concepto de lo que se queria para la pagina claro que habia que ser realistas que no se podian pedir cosas exajeradas teniendo en cuenta que todavia soy un programador junior pero podia hacer cosas y eso sirvio de algo para este proyecto

**CREACION DEL CODIGO**
El codigo fue una locura al principio no sabia por donde empezar pero queria hacer de todo despues de organizar prioridades empece por partes el menu, el banner, el contenido y el pie de pagina claramente todo esto fue hecho con bastante cuidado asegurandome de que el codigo quedara lo mas organizado visualmente al principo era un desorden visual (lo que yo llamo codigo crudo) despues de hacer que todo funcionara perfectamente lo organice (lo que yo llamo codigo limpio) y es el que es publicado hoy en este repositorio
