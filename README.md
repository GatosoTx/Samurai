# **¡IMPORTANTE LEER ANTES DE VER LA PAGINA Y EL CODIGO!**
> *Esta página web es un proyecto para el área de tecnología de mi colegio. No soy tan experto, así que es común encontrar el código un poco desordenado y variables con nombres poco específicos. El código se encuentra en desarrollo todavía; es de esperar que ciertas cosas no estén como lo planteado y que otras, bueno, no funcionen como deberían. A continuación, una explicación del código y cómo fue su realización.*

**PLANTIAMIENTO DE LA IDEA**

La idea era una página que reflejara la rebeldía que representa la banda. La idea se plasmó en un wireframe de baja y alta fidelidad, en los cuales se creó un concepto de lo que se quería para la página; claro que había que ser realistas, que no se podían pedir cosas exageradas teniendo en cuenta que todavía soy un programador junior, pero podía hacer cosas y eso sirvió de algo para este proyecto.

**CREACION DEL CODIGO**

El codigo fue una locura al principio no sabia por donde empezar pero queria hacer de todo después de organizar prioridades empece por partes el menu, el banner, el contenido y el pie de pagina claramente todo esto fue hecho con bastante cuidado asegurandome de que el codigo quedara lo mas organizado visualmente al principo era un desorden visual (lo que yo llamo codigo crudo) después de hacer que todo funcionara perfectamente lo organice (lo que yo llamo codigo limpio) y es el que es publicado hoy en este repositorio el codigo esta hecho en html5 y css

**ASPECTO VISUAL**

El css es algo sublime en mi opinion lo realize con bastente esmero es la primera vez que hacia un codigo tan extenso y complejo para el ojo inexperto es simplemete un monton de letras y numeros pra el ojo experto en cambio es un codigo decente que cumple la necesidad de la pagina y claro podria mejorar (y lo hara) opte por colores representativos de la banda los cuales eran el blanco, el rojo y el negro a su vez también opte por que la pagina tuviera un diseño perecido a lo que busca transmitir la banda
